# hexiaopi.github.io 何小进童鞋个人博客

## 功能特性
> 记录个人开发编程的博客

## 快速开始

### 依赖检查

> 检查本地是否安装docsify

```bash
docsify -v
# docsify-cli version:
#  4.4.3
```

### 运行
```bash
docsify serve docs
# Serving /Users/hexiaopi/hexiaopi.github.io/docs now.
# Listening at http://localhost:3000
```
> 浏览器打开http://localhost:3000

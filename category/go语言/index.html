        深入了解Go锁原理

        Go语言sync包提供了两种锁：

        • 互斥锁 Mutex:
          • 加锁时，其他加锁协程均阻塞
        • 读写锁 RWMutex:
          • 加读锁时，加写锁协程阻塞，加读锁协程不阻塞
          • 加写锁时，其他加读写协程均阻塞

        小心break🤦‍♂️

        这里展示break的一个例子

        package main

import (
	"fmt"
	"time"
)

func main() {
	exit := make(chan interface{})

	go func() {
		for {
			select {
			case <-time.After(time.Second):
				fmt.Println("tick")
			case <-exit:
				fmt.Println("exiting...")
				break
			}
		}
		fmt.Println("exit!")
	}()

	time.Sleep(3 * time.Second)
	exit <- struct{}{}

	// wait child goroutine exit
	time.Sleep(3 * time.Second)
}

recover注意事项

Hexiaopi小于 1 分钟编程语言Go基础
此页内容
  • 正确使用示例
  • 错误用例：不使用defer
  • 错误用例：recover 在 defer 嵌套函数中
  • defer recover 再次panic

defer结合recover通常用于捕获panic

正确使用示例

func PrintRecover() {
	defer func() {
		if err := recover(); err != nil {
			fmt.Println(err)
		}
	}()
	fmt.Println("a")
	panic("some thing")
	fmt.Println("b")
}

使用示例

func ExamplePrintRecover() {
	PrintRecover()
	// Output:
	// a
	// some thing
}

错误用例：不使用defer

func FailRecoverWithoutDefer() {
	if err := recover(); err != nil {
		fmt.Println(err)
	}
	fmt.Println("a")
	panic("some thing")
	fmt.Println("b")
}

警告

不使用defer 方式，将无法捕获异常，程序陷入panic

错误用例：recover 在 defer 嵌套函数中

func FailRecoverWithoutLevelDefer() {
	defer func() {
		func() { //recover 在 defer 嵌套函数中
			if err := recover(); err != nil {
				fmt.Println(err)
			}
		}()
	}()
	fmt.Println("a")
	panic("some thing")
	fmt.Println("b")
}

警告

defer recover和panic不在同一层级，导致无法捕获异常，程序陷入panic

defer recover 再次panic

package main

import "fmt"

func main() {
	defer func() {
		fmt.Println("xx", recover())
	}()
	defer func() {
		defer func() {
			fmt.Println("yy", recover())
		}()
		panic(1)
	}()
	defer recover()
	panic(2)
}
执行结果
yy 1
xx 2
